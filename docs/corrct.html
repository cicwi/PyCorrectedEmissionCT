Creates a synthetic sinogram, from the given volume, attenuations and PSF.

  • vol (numpy.array_like) – Volume containing the elemental concentrations or other quantities.

  • angles_rad (numpy.array_like) – Angles in radians of each sinogram line

  • vol_att_in (numpy.array_like) – Volume containing the local attenuation coefficients for the incoming photons

  • vol_att_out (numpy.array_like) – Volume containing the local attenuation coefficients for the emitted photons

  • angles_detectors_rad (numpy.array_like or float, optional. Deafult: numpy.pi / 2) – Detectors’ position angles in radians

  • weights_detectors (numpy.array_like or float, optional. Deafult: None) – Detectors’ weights

  • psf (numpy.array_like, optional. Deafult: None) – Detector point spread function (PSF)

  • data_type (numpy.dtype, optional. Default: numpy.float32) – Volume data type

Returns

The simulated sinogram

corrct.corrct.reconstruct(algo, sino, angles_rad, iterations=None, vol_att_in=None, vol_att_out=None, angles_detectors_rad=1.5707963267948966, weights_detectors=None, lower_limit=None, upper_limit=None, apply_circ_mask=True, symm=True, lambda_reg=0.01, data_term='l2', psf=None, data_type=<class 'numpy.float32'>)[source]

Reconstructs the given sinogram, with the requested algorithm.

Parameters

  • algo (string. Options: 'SART' | 'SIRT' | 'CP' | 'CPTV' | 'CPL1' | 'CPWL') – Reconstruction algorithms to use.

  • sino (numpy.array_like) – The sinogram to recosntruct

  • angles_rad (numpy.array_like) – Angles in radians of each sinogram line

  • iterations (int, optional. Default: None) – Number of iterations

  • vol_att_in (numpy.array_like, optional. Default: None) – Volume containing the local attenuation coefficients for the incoming photons

  • vol_att_out (numpy.array_like, optional. Default: None) – Volume containing the local attenuation coefficients for the emitted photons

  • angles_detectors_rad (numpy.array_like or float, optional. Default: numpy.pi / 2) – Detectors’ position angles in radians

  • weights_detectors (numpy.array_like or float, optional. Default: None) – Detectors’ weights

  • lower_limit (float, optional. Default: None) – Lower clipping limit

  • upper_limit (float, optional. Default: None) – Upper clipping limit

  • apply_circ_mask (boolean, optional. Default: True) – Switch to activate a circular volume mask

  • symm (boolean, optional. Default: True) – Switch to define whether the projectors should be symmetric

  • lambda_reg (float, optional. Default: 1e-2) – Regularizer weight

  • data_term (string, optional. Options: 'l2' | 'kl'. Default: 'l2') – Data fidelity term

  • psf (numpy.array_like, optional. Default: None) – Detector point spread function (PSF)

  • data_type (numpy.dtype, optional. Default: numpy.float32) – Volume data type

Raises

ValueError – Raises an error if the algorithm is not known.

Returns

The reconstructed volume

Return type
